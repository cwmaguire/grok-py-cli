# Grok CLI Python Documentation

Welcome to the documentation for Grok CLI Python implementation.

## Overview

Grok CLI is an AI-powered terminal assistant that allows you to interact with Grok (xAI's AI model) directly from your command line. This Python implementation provides the same functionality as the original TypeScript version with additional Python ecosystem benefits.
